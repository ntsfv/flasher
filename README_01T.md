## UART загрузчик для микроконтроллера К1921ВК01Т

Описание протокола находится в ```NOTES.md```.

### Описание

* Загрузчик работает через UART при параметрах: 8 бит, без бита четности
* Скорость UART определяется автоматически при инициализации соединения
* Загрузчик запускается лишь при активном уровне на BOOTEN (лог. единица) при сбросе (должен быть установлен в неактивное состояние после инициализации соединения)
* Системная частота 100 МГЦ (выход PLL, работающей от внешнего источника)
* Загрузчик позволяет принимать непрерывный поток данных - объем внутреннего FIFO 64 КБ
* Реализованы команды:
    - запрос информации о микроконтроллере и загрузчике
    - запрос CFGWORD
    - модификация CFGWORD (при сохранении остальной информации на странице)
    - чтение страницы BootFlash, Userflash(как основной области, так и NVR)
    - запись страницы BootFlash, Userflash(как основной области, так и NVR)
    - стирание страницы BootFlash, Userflash(как основной области, так и NVR)
    - стирание всей основной области BootFlash, Userflash
* Опционально, запись страницы можно проводить с её предварительным стиранием
* Существует возможность запретить чтение страниц BootFlash, Userflash (как основной области, так и NVR). Для этого CFGWORD расширен - загрузчик дополнительно задействует 17, 18, 25, 26 биты слова.
* Команда модификации CFGWORD производит контроль записанной информации
* Загрузчик не позволяет осуществлять чтение, стирание или запись страниц, где он расположен

### Конфигурация

Загрузчик должен быть сконфигурирован под необходимую плату/аппартное окружение. Конфигурация производится с помощью дефайнов в ```firmware/k1921vk01t/CmakeLists.txt```. Подробнее в ```firmware/k1921vk01t/README.md```

### Подключение

Вывод BOOTEN может быть любым пином на любом порту, в зависимости от конфигурации.
Вывода UART зависят от того, какой UART в конфигурации выбран.

|  UART |     RX    |     TX    |
|:-----:|:---------:|:---------:|
| UART0 | GPIOС[4]  | GPIOC[3]  |
| UART1 | GPIOA[4]  | GPIOA[3]  |
| UART2 | GPIOF[11] | GPIOF[10] |
| UART3 | GPIOD[1]  | GPIOD[2]  |

Непосредственно само подключение к USB-UART переходнику:

| К1921ВК01T | USB-UART |
|:----------:|:--------:|
| UART TX    | RX       |
| UART RX    | TX       |
| BOOTEN     | RTS      |
| RST        | DTR      |

```RTS``` и ```DTR``` подключать не обязательно, но рекомендуется, т.к. это позволит автоматически управлять сбросом микроконтроллера и входом в режим загрузчика.
В противном случае последовательность действий следующая:

* Подключаем ```RX``` и ```TX```
* Подтягиваем ```BOOTEN``` к питанию
* Сбрасываем микроконтроллер (```RST``` к земле)
* Подключаемся через ПО
* Ждём пока появится сообщение о том, что устройство подключено
* Снимаем подтяжку с ```BOOTEN```
* Выполняем дальнейшие операции
