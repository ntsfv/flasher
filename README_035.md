## UART загрузчик для микроконтроллера 1921ВК035

Описание протокола находится в ```NOTES.md```.

### Описание

* Загрузчик работает через UART1 при параметрах: 8 бит, без бита четности
* Скорость UART определяется автоматически при инициализации соединения
* Загрузчик запускается лишь при активном NMI (лог. ноль) при сбросе (NMI должен быть установлен в неактивное состояние после инициализации соединения)
* Системная частота 100 МГЦ (выход PLL, работающей от внутреннего источника 8 МГц)
* Загрузчик позволяет принимать непрерывный поток данных - объем внутреннего FIFO 8 КБ
* Реализованы команды:
    - запрос информации о микроконтроллере и загрузчике
    - запрос CFGWORD
    - модификация CFGWORD (при сохранении остальной информации на странице)
    - чтение страницы (как основной области, так и NVR)
    - запись страницы (как основной области, так и NVR)
    - стирание страницы (как основной области, так и NVR)
    - стирание всей основной области
* Опционально, запись страницы можно проводить с её предварительным стиранием
* Существует возможность запретить чтение страниц (как основной области, так и NVR). Для этого CFGWORD расширен - загрузчик дополнительно задействует 7, 6 биты слова.
* Команда модификации CFGWORD производит контроль записанной информации
* Загрузчик не позволяет осуществлять чтение, стирание или запись страниц, где он расположен

Когда загрузчик зашит и активирован - микроконтроллер по включению питания всегда проходит через него. Если на момент включения был активен NMI, то осуществится переход к исполнению тела загрузчика и ожидание связи по UART. Если NMI был неактивен, то с помощью записи в регистр контролера флэша BDIS и программного сброса ядра произойдёт переход к исполнению пользовательской программы c 0x00000000.
Дополнительные требования к линковке пользовательской программы отсутствуют.

# Сборка и загрузка

Требуется:

* CMake 3.8+
* make
* openocd

```bash
# -- Сборка
cd firmware/k1921vk035
mkdir build-release && cd build-release
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
# -- Загрузка
cd ../init
# Разрешаем работу загрузчика
openocd -f enable_boot.cfg
# Выполняем сброс по питанию или по кнопке, затем зашиваем загрузчик с помощью
openocd -f program_boot.cfg
```

### Подключение

|  К1921ВК035   | USB-UART |
|:-------------:|:--------:|
| UART1 TX (B8) | RX       |
| UART1 RX (B9) | TX       |
| NMI           | RTS      |
| nRST          | DTR      |

```RTS``` и ```DTR``` подключать не обязательно, но рекомендуется, т.к. это позволит автоматически управлять сбросом микроконтроллера и входом в режим загрузчика.
В противном случае последовательность действий следующая:

* Подключаем ```RX``` и ```TX```
* Подтягиваем ```NMI``` к земле
* Сбрасываем микроконтроллер (```nRST``` к земле)
* Подключаемся через ПО
* Ждём пока появится сообщение о том, что устройство подключено
* Снимаем подтяжку с ```NMI```
* Появится сообщение о том, что устройство готово
* Выполняем дальнейшие операции
