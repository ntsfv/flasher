## UART загрузчик для микроконтроллера К1921ВК028

Описание протокола находится в ```NOTES.md```.

### Описание

* Загрузчик работает через UART при параметрах: 8 бит, без бита четности
* Скорость UART определяется автоматически при инициализации соединения
* Системная частота - выход PLL, работающей от SRCCLK
* Вход в загрузчик, зашитый в BFLASH, осуществляется путём подтяжки к питанию вывода BOOT0
* Загрузчик позволяет принимать непрерывный поток данных - объем внутреннего FIFO 128 КБ
* Реализованы команды:
    - запрос информации о микроконтроллере и загрузчике
    - запрос CFGWORD
    - модификация CFGWORD (при сохранении остальной информации на странице)
    - чтение страницы MFLASH, BFLASH (как основной области, так и NVR)
    - запись страницы MFLASH, BFLASH (как основной области, так и NVR)
    - стирание страницы MFLASH, BFLASH (как основной области, так и NVR)
    - стирание всей основной области MFLASH, BFLASH
* Опционально, запись страницы можно проводить с её предварительным стиранием
* Существует возможность запретить чтение страниц MFLASH, BFLASH (как основной области, так и NVR). Для этого CFGWORD расширен - загрузчик дополнительно задействует 12-15 биты слова CFGWORD1.
* Команда модификации CFGWORD производит контроль записанной информации
* Загрузчик не позволяет осуществлять чтение, стирание или запись страниц, где он расположен

### Конфигурация

Прошивка конфигурируется с помощью дефайнов, заданных в скрипте ```firmware/k1921vk028/CMakeLists.txt```

Номер UART может принимать значения: 0 - 3. По умолчанию, выбран UART0.
```
ADD_DEFINITIONS(-DUART_NUMBER=0)
```

Вывода UART зависят от того, какой UART в конфигурации выбран.

|  UART |     RX    |     TX    |
|:-----:|:---------:|:---------:|
| UART0 | GPIOС[11] | GPIOC[10] |
| UART1 | GPIOC[15] | GPIOC[14] |
| UART2 | GPIOF[9]  | GPIOC[8]  |
| UART3 | GPIOE[8]  | GPIOE[9]  |

Частота внешнего генератора или кварцевого осциллятора может принимать значения: 8000000, 12000000, 16000000, 20000000, 24000000
```
ADD_DEFINITIONS(-DOSECLK_VAL=12000000)
```

### Сборка и загрузка

Требуется:

* CMake 3.8+
* make
* openocd

```bash
# -- Сборка
cd firmware/k1921vk028
mkdir build-release && cd build-release
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
# -- Загрузка
cd ../init
# Разрешаем работу загрузчика, подтянув BOOT0 к питанию.
# Выполняем сброс по питанию или по кнопке, затем зашиваем загрузчик с помощью
openocd -f program_boot.cfg
```

### Подключение

Подключение к USB-UART переходнику:

| К1921ВК028 | USB-UART |
|:----------:|:--------:|
| UART TX    | RX       |
| UART RX    | TX       |
| BOOT0      | RTS      |
| RST        | DTR      |

```RTS``` и ```DTR``` подключать не обязательно, но рекомендуется, т.к. это позволит автоматически управлять сбросом микроконтроллера и входом в режим загрузчика.
В противном случае последовательность действий следующая:

* Подключаем ```RX``` и ```TX```
* Подтягиваем ```BOOT0``` к питанию
* Сбрасываем микроконтроллер (```RST``` к земле)
* Подключаемся через ПО
* Ждём пока появится сообщение о том, что устройство подключено
* Снимаем подтяжку с ```BOOT0```
* Выполняем дальнейшие операции
* По отключению и сбросу будет запущена пользовательская прошивка
